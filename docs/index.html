说明

本 demo 使用 @jswork/hash-query 管理 URL hash 中的查询参数。

例如：地址栏为 ...#/app?lang=zh&debug=true，则管理 ?lang=zh&debug=true 部分。

支持方法：

  • hq.get() → 获取 URLSearchParams
  • hq.set(params, replace?) → 全量替换
  • hq.update({ key: value }, replace?) → 增量更新（传 null 删除）
  • hq.toJson() → 转为普通对象
